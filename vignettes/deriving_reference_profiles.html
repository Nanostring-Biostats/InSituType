Deriving reference profiles from scRNA-seq data

Here we’ll show how to derive reference profiles from a scRNA-seq dataset. These profiles can then be used to run Insitutype in supervised or semi-supervised mode.

First, we load the package and access the example data.

library(InSituType)
data("mini_nsclc")

You’ll have a scRNA-seq dataset in whatever format you prefer. For our tools, you’ll just need to extract the raw, linear-scale expression matrix and a vector of cell type assignments.

# pretend the demo cosmx data is your scRNA-seq counts matrix
counts <- mini_nsclc$counts  
dim(counts)  # cells in rows, genes in columns
#> [1] 2198  960
counts[1:5,1:5]  # see linear scale
#>          AATK ABL1 ABL2 ACE ACE2
#> c_3_18_2    0    0    0   0    0
#> c_3_18_3    0    0    0   0    0
#> c_3_18_4    0    0    0   0    0
#> c_3_18_5    0    0    0   0    0
#> c_3_18_6    0    0    0   0    0

# make up a vector of cell type assignments:
clust <- sample(c("a","b","c"), nrow(counts), replace = TRUE)
table(clust)
#> clust
#>   a   b   c 
#> 748 710 740

Now run the getRNAprofiles function:

profiles <- getRNAprofiles(x = counts, 
                           neg = 0,   # assume no background in scRNA-seq data. 
                           clust = clust) 
head(profiles)
#>                c           a          b
#> AATK  0.01081081 0.009358289 0.01126761
#> ABL1  0.02702703 0.029411765 0.03380282
#> ABL2  0.05675676 0.070855615 0.06197183
#> ACE   0.02972973 0.049465241 0.02676056
#> ACE2  0.02432432 0.026737968 0.03239437
#> ACKR1 0.10135135 0.081550802 0.07887324

And of course, if you were deriving a profile matrix from a previous CosMx dataset or any other technology with background, you’d want to enter the per-cell background level in the “neg” argument.
